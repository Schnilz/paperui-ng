What openHAB can do for you

openHAB is a mature home automation platform that is protocol agnostic, meaning it can connect to nearly any home automation hardware on the market today.

You need to extend openHAB with additional software, called Add-ons, via this web interface. Add-ons add device & services feature detection and moments later you can already control your external solution.

Via scheduled tasks and rules you start automating your installation. For instance turning your lights on when your phone is in WiFi range, but only if it is already later than 6 pm. Only your imagination is the limit.

openHAB also and supports DIY and moddable hardware. We speak about this later on.

About this interface

Use this interface to setup new devices, configure existing ones, create automation rules and timers.

Every section (except this tutorial) has a context help. Click on the question mark that appears next to the title.

You do not control your devices via this interface. Download the Android App, iOS App, HaBot or use HabPanel instead. The last section of this tutorial deals with the different control interfaces.
