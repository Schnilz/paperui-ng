About this interface

This interface helps you to configure and maintain your openHAB 2.5 and newer installation.

How can I get involved?

Please post texts, pictures, videos of your ideas on how to improve this interface to the Github Issue section of this project.

Please always use the latest browser version, before you report any problems, thanks.
